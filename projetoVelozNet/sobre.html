Conheça a VelozNet

Este site foi criado por Hagatha Byatriz Ribeiro da Silva, desenvolvido ultilizando apenas HTML, CSS e JavaScript, inspirado em uma ampla pesquisa de diversos templates disponíveis na internet, bem como no conteúdo absorvido ao longo das semanas durante a disciplina de Fundamentos de Programação Web.

A plataforma proporciona informações sobre planos de Internet de alta velocidade, com detalhes claros permitindo que o usuário escolha o plano que melhor atende ás suas necessidades.

O site é composto por:

Página Home - página inicial do site onde são apresentados os produtos. Composta por uma barra de navegação fixa com links que para acesso as demais páginas do site, sessão de produtos com links direto para a página de cadastro e rodapé também com links para facilitar a navegação pelo site.

Página Sobre - página de explicação do site. Contém informações sobre o site, autor e detalhes sobre as páginas.

Formuário de cadastro - página onde o usuário adquire o plano desejado. Contém campos para o usuário inserir suas informações pessoais, escolher o plano e aceitar os termos de serviço. Os dados inseridos são verificados atravéz do código JavaScript.

Página de confirmação - página para confirmar os dados inseridos no fomulário. São mostrados os dados que o usuário digitou anteriormente no formulário.

Todas as páginas possuem uma barra de navegação e um rodapé com links para navegar pelo site, facilitando a experiência do usuário.
